13. Puzzle (if time permits)

Make a function overload(function_with_n_parameters, function_with_m_parameters) that:
  1. throws a TypeError, if both function have the same number of parameters,
  2. and otherwise, returns an function that
  3. when called with n parameters, forwards the call to function_with_n_parameters,
  4. when called with m parameters, forwards the call to function_with_m_parameters,
  5. when called otherwise, throws a TypeError
  6. (hint: every function has a length property that returns the number of formal paramters in the definition of that function)
var createVector = overload( function (a, b) { return { x: a, y: b }; }, function (length) { return { x: length / 1.414, y: length / 1.414 }; } ); createVector(3, 4) // => { x: 3, y: 4 } createVector(7.07) // => { x: 5, y: 5 }
