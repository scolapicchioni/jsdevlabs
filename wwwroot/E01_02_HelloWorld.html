Lab 1 – Introduction

1. Install Browser

Install the Google Chrome Browser (recommended) on your machine (if it is not already there) and make it your default browser.

2. Write Hello World

  • Create two files:
    • Hello World.html
    • Hello World.js
    1. In Hello World.js, write a JavaScript function that prints “Hello World” in the console
    2. In Hello World.html, make a reference to Hello World.js and make sure the function is called
    3. Open Hello World.html in Chrome, open the console (F12) and observe the result
    4. Declare a variable: me
    5. Assign an object literal to me that holds your name and city
    6. Print it in the console
